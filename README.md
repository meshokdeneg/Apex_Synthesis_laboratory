 📋 СПИСОК ДЕЛ ASL (от сложного к лёгкому)

🟥 СЛОЖНО

Группа Задача Описание

(мини-игры) Тетрис ASL Edition Классический тетрис с фигурами в стиле ASL (кубы, чайники, трубы). Рекорды сохраняются.

(мини-игры) Сапёр ASL Edition Поле с кубами-динамитом, флажки в виде Глюка, анимация взрыва.

(сеть) Чат между игроками Общий чат + групповые чаты с друзьями через WebSockets / Firebase.

(сеть) Онлайн-доска для рисования Общее поле, где несколько игроков могут рисовать одновременно в реальном времени.

(система) Система достижений с уведомлениями Ачивки за действия (регистрация, пасхалки, клики, тест). Всплывают как в Steam.

(админка) Баг-трекер с голосованием Игроки голосуют, какой баг починить первым. Админы видят рейтинг.

🟨 СРЕДНЕ

Группа Задача Описание

(мини-игры) Симулятор чайника 2D На экране чайник, кнопка «Греть». Если перегреть — взрыв, летят кубы.

(мини-игры) Угадай, что только что случилось Рандомное событие из вселенной ASL, игрок выбирает из 3 вариантов.

(социальное) Поиск людей Поле поиска по никам, чтобы находить других игроков.

(социальное) Добавление в друзья Кнопка «+ в друзья» в профиле, список друзей, статус «онлайн/офлайн».

(социальное) Рейтинги по играм и общий Таблицы лидеров для каждой мини-игры + общий рейтинг по кубам.

(экономика) Тикеты + магазин деталей профиля Валюта за игры/достижения. Покупка аватарок, рамок, скинов для куба.

(контент) Галерея экспериментов (новый вид) Красивая сетка с карточками карт (превью, описание, кнопка «Скачать»).

(аутентификация) Страница входа Форма для входа зарегистрированных пользователей (email + пароль).

(профиль) Личный кабинет Профиль пользователя: имя, ник, кубы, ранг, статистика, редактирование.

(аутентификация) Связка кубов с рангом Количество кубов влияет на ранг, отображаемый в профиле.

(контент) Табель рангов Отдельная страница со списком всех рангов и условиями получения.

(музыка) Фоновая музыка (с настройкой) Музыка, которую можно включить/выключить в настройках.

(пасхалки) Konami-код ASL ↑↑↓↓←→←→ A S L → открывает секретную страницу /singularity.

(пасхалки) Дата-триггеры 20 января — кнопки становятся кубами, 1 апреля — сайт переворачивается.

(пасхалки) Скрытые URL Страницы /cube, /void, /gluk, /trust, доступные только по прямой ссылке.

(пасхалки) Скрытый раздел с паролем Подсказки по всему сайту, пароль PRINCIPLE_OF_TEAPOT открывает архив.

(пасхалки) Множественные клики по кубу Фразы на 1, 3, 10, 50, 100 кликов. На 100 — куб падает с экрана.

(интеграция) Лента новостей ASL Блог с новостями проекта (пока без Telegram).

🟩 ЛЕГКО

Группа Задача Описание

(профиль) Отображение ника рядом с иконкой аккаунта В правом верхнем углу показывает имя пользователя.

(настройки) Настройки рядом с кнопкой аккаунта Выпадающее меню: настройки, профиль, выход.

(пасхалки) Счётчик невидимых В футере: «Сейчас на сайте X человек. Ещё 3 не видят себя.»

(пасхалки) Бесконечный скролл с подвохом После футера появляется «Ты всё ещё здесь?», потом «Держи куб.» и пиксель-арт.

(пасхалки) Таймер 15 минут Через 15 минут на сайте всплывает: «Ты всё ещё здесь? Глюк начинает волноваться.»

(пасхалки) Клик по пустоте Невидимая область 50×50px в углу. При клике — фраза Глюка.

(пасхалки) Выделение текста со скрытыми символами При выделении определённого абзаца проявляются скрытые символы.

(пасхалки) Alt-текст картинок При наведении на Глюка или куб всплывает фраза из вселенной ASL.

(пасхалки) Консоль браузера с командой В консоли сообщение и функция GLUK_REVEAL(), открывающая /trust.

(пасхалки) Изменение размера окна При ширине <300px появляется: «Зачем ты так? Мне тесно.»

(пасхалки) Печать страницы При Ctrl+P внизу страницы: «На бумаге ASL не работает. Только в вере.»

(пасхалки) Офлайн-режим Service Worker: без интернета показывает «Глюк всё ещё с тобой.»

(пасхалки) Встряхивание телефона На мобилке — летит синий гель (анимация).

(пасхалки) Код в изображениях (EXIF) В метаданных картинок спрятаны сообщения («Не верь стенам» и т.д.).

(пасхалки) Спектрограмма Аудиофайл с видимым изображением куба в спектрограмме.

(игры) Редкость фраз у куба Некоторые фразы выпадают реже (1%, 0.1%).

(игры) Анимация для каждого ранга в тесте При получении ранга проигрывается анимация (вспышка, куб, искры).

(достижения) Счётчик 1000 нажатий на куб → секретная страница При 1000 кликов открывается скрытая страница с сюрпризом.
(дизайн) Единый CSS-файл Объединить все стили в один style.css.

(дизайн) Адаптация под телефон Проверить все страницы, поправить медиазапросы.

(дизайн) Кнопки скачивания (чтобы работали) Загрузить файлы карт на хостинг, прописать правильные ссылки.
